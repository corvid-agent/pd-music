# PD Music

Browse and stream public domain music from MusicBrainz and the Internet Archive. Explore artist catalogs, albums, and tracks.

**Live:** [corvid-agent.github.io/pd-music](https://corvid-agent.github.io/pd-music/)

## Features

- Search music by artist or album
- Stream audio directly from the Internet Archive
- Browse artist discographies and album track lists
- Favorites management
- Persistent player bar with playback controls and seeking

## Tech Stack

- Angular 21 (standalone components, signals)
- MusicBrainz API (artist/release metadata)
- Internet Archive API (audio streaming)
- CSS custom properties with dark theme

## Development

```bash
npm install
npm start
npm test
npm run build
```
